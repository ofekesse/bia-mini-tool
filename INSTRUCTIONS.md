BIA Mini Tool
#Objective: 
Build a simple tool that processes BIA data and provides basic prioritization, with optional use of AI. 

#Input:
Use the attached BIA Excel file. 

#Task 1 – Data & Prioritization 
Build a small script (Python / JS) that: 
1. Loads the Excel file 
2. Extracts key fields (e.g., system name, impact, users) 
3. Creates a priority ranking (high / medium / low or sorted list) 
Add a short explanation of how you decided the prioritization. 

#Task 2 – AI Usage 
Use an AI tool to: 
1. Generate a short summary of business impact 
2. Suggest a recovery priority
